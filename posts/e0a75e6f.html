Node.js安装配置

发表于 2018-12-31 | 更新于 2019-06-03 | 分类于 开发环境

Window10中Node.js的安装配置。

  1. 下载，建议下载LTS版本
  2. 安装程序，建议勾选Add to PATH，新版本中已经默认勾选
  3. 安装验证：node -v查看安装的nodejs版本号，npm config ls查看配置信息
  4. 修改全局安装路径，默认是安装至C盘用户目录下的，改为默认保存至nodejs的安装目录（这里为D:\nodejs）。执行命令npm config set prefix "D:\nodejs"。nodejs会自动寻找该路径下的node_modules文件夹为实际存放全局模块的路径，这也是为啥叫prefix不叫global的原因；以后安装的全局模块都会被放到D:\nodejs\node_modules下，跟默认内置的npm模块在一个文件夹中
  5. 修改cache路径：npm config set cache "D:\nodejs\node_cache"
  6. 上面第4和第5点的配置修改后会在C盘用户目录下生成.npmrc文件，即配置信息，若上述方法修改后无效果，可以直接修改D:\nodejs\node_modules\npm\npmrc中的内容

本文环境

操作系统：Windows 10 64位
